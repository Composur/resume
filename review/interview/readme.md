### CSS
####  盒模型
+ content-box:width=内容区宽度
+ border-box:width=内容区宽度+padding宽度
#### css reset和normalize.css的区别
+ reset重置样式，抛弃之前的样式
+ normalize让所有的浏览器的标签都根规定的默认样式一致（Linux、window平台下）
